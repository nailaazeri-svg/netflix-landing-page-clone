# 🎬 Netflix Landing Page UI Clone

A responsive Netflix landing page UI clone built using **HTML5** and **CSS3**.  
This project replicates the Netflix homepage design while focusing on layout, responsiveness, and clean UI structure.

## 🚀 Live Demo

🔗 https://nailaazeri-svg.github.io/netflix-landing-page-clone/

## 🛠 Technologies Used

- HTML5
- CSS3
- Flexbox
- Responsive Design

## 📱 Features

- Responsive layout
- Netflix-style hero section
- FAQ section
- Mobile friendly design
